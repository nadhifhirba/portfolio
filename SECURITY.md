# Security Policy

## Reporting a Vulnerability

If you find a security vulnerability in this site, please email **nadhifhirba@gmail.com** directly rather than opening a public issue.

Include a description of the issue and steps to reproduce. I'll respond within 5 business days.

## Scope

- XSS, injection, or CSP bypass on nhadesign.com
- Security header misconfigurations

Out of scope: spam, social engineering, issues in third-party services (Vercel, Google Fonts, Cloudflare).
